# Mongodb Challenge

## Getting Started

### Installation

1. Clone the repository.
### git clone https://github.com/sonalikari/challenge-with-mongodb.git

2. Run the command for backend.
### cd backend
### npm install
### npm start

3. Run the command for frontend.
### cd frontend
### npm install
### npm start

4. Then on browser open the following links:-
